## Claude Code Skills

Install with `npx skills add <repo>`. Browse all at [skills.sh](https://skills.sh/).

| Skill | Install |
|-------|---------|
| **InfAU skills** | `npx skills add bauhaus-infau/skills` |
| **frontend-design** | `npx skills add anthropics/skills` |
| **impeccable** | `npx skills add pbakaus/impeccable` |
| **shadcn** | `npx skills add shadcn/ui` |
| **React best practices** | `npx skills add vercel-labs/agent-skills` |

### Useful Slash Commands

| Command | What it does |
|---------|--------------|
| `/help` | Show help and available commands |
| `/clear` | Clear conversation and start fresh |
| `/diff` | View uncommitted changes |
| `/rewind` | Undo — rewind conversation and code |
| `/compact` | Compress conversation to save context |

## Free Data Sources

| Source | Link | Description |
|--------|------|-------------|
| **Overpass Turbo** | [overpass-turbo.eu](https://overpass-turbo.eu/) | Query + export OSM data as GeoJSON |
| **Overture Maps** | [overturemaps.org](https://overturemaps.org/download/) | 2.6 billion buildings worldwide |
| **basemap.de** | [basemap.de](https://basemap.de/produkte-und-dienste/) | Official German basemap — WMS/WMTS, vector tiles, 3D |

| **Open-Meteo** | [open-meteo.com](https://open-meteo.com/) | Weather, elevation, air quality — no API key needed |
