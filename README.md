# AI Lab - InfAU Workshop

A hands-on workshop where Bauhaus InfAU staff use AI-assisted coding (Claude Code or OpenAI Codex CLI) to build an isovist web app — no prior web development experience needed.

Everyone builds the same baseline: a 3D Weimar city view with isovist visualization using Three.js. Then each participant customizes and extends it their own way.

## What's Inside

| Folder        | Contents                                                                               |
| ------------- | -------------------------------------------------------------------------------------- |
| `workshop/` | Workshop guide, starter prompt, cheatsheet, and Weimar GeoJSON data                    |
| `results/`  | Example projects and shared gallery                                                        |

## Getting Started

See the full workshop guide: [`workshop/README.md`](workshop/README.md)

**Prerequisites:**

1. [VS Code](https://code.visualstudio.com/download)
2. [Claude Pro or Max](https://claude.ai/pricing) subscription (or OpenAI API key for Codex CLI)
3. [GitHub account](https://github.com/signup) (free)

## Results

Each participant deploys their own app under the `bauhaus-infau` org at `https://bauhaus-infau.github.io/isovist-<name>/`.
